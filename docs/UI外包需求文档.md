# 海澄村三色资源游戏UI外包需求文档

## 一、项目概述

本项目是一个基于微信小游戏平台的游戏化小程序，旨在通过游戏的形式展示海澄村的非遗、自然、红色三色资源，提升用户对海澄村文化的认知和了解。项目包含三个核心游戏：消消乐、三色拼图和三色答题。

## 二、游戏类型和功能

### 2.1 消消乐游戏
- **游戏玩法**：通过交换相邻的方块，形成3个或以上的连续匹配，消除方块获得分数
- **核心功能**：
  - 8×8游戏棋盘
  - 三种资源类型（非遗、自然、红色）的方块
  - 特殊方块（行消除、列消除、彩虹球）
  - 计分系统和等级系统
  - 倒计时功能
  - 游戏结束判定

### 2.2 三色拼图游戏
- **游戏玩法**：通过移动拼图块，将打乱的图片恢复完整
- **核心功能**：
  - 3×3/4×4/5×5三种难度的拼图
  - 三种资源类型（非遗、自然、红色）的拼图图片
  - 计时功能
  - 提示功能
  - 拼图完成后的打卡海报生成

### 2.3 三色答题游戏
- **游戏玩法**：回答关于海澄村三色资源的问题，获得分数
- **核心功能**：
  - 三种难度等级的题目
  - 三种资源类型（非遗、自然、红色）的题目
  - 计时功能
  - 提示功能
  - 答题结果反馈
  - 答题完成后的成绩统计

## 三、UI设计规范

### 3.1 整体风格
- **设计风格**：卡通化、简洁明了、色彩鲜艳
- **目标受众**：全年龄段用户，重点面向青少年和年轻成人
- **设计理念**：融合海澄村的文化元素，突出三色资源的特色

### 3.2 色彩方案

#### 3.2.1 主色调
- **非遗**：#FF5722（橙色）
- **自然**：#4CAF50（绿色）
- **红色**：#F44336（红色）

#### 3.2.2 辅助色
- **背景色**：#F5F5F5（浅灰色）
- **文字色**：#333333（深灰色）
- **按钮色**：
  - 主按钮：#2196F3（蓝色）
  - 成功按钮：#4CAF50（绿色）
  - 警告按钮：#FF9800（橙色）
  - 次要按钮：#9E9E9E（灰色）

### 3.3 界面布局

#### 3.3.1 消消乐游戏
- **顶部信息栏**：显示得分、等级、时间
- **中部游戏区域**：8×8游戏棋盘
- **底部功能区**：返回按钮、重新开始按钮
- **游戏结束界面**：显示最终得分、等级，提供重新开始和返回首页按钮

#### 3.3.2 三色拼图游戏
- **顶部信息栏**：显示拼图名称、计时、难度标识
- **中部游戏区域**：左侧目标图案预览，右侧拼图区域
- **底部功能区**：提示按钮、难度选择按钮、重置按钮、分享按钮
- **拼图完成界面**：显示用时和难度等级，提供再玩一次、查看科普、分享海报、返回首页按钮

#### 3.3.3 三色答题游戏
- **顶部信息栏**：显示题目序号、计时、资源类别标识
- **中部内容区域**：题目内容和选项区域
- **底部功能区**：提示按钮、提交按钮、跳过按钮
- **答题结果界面**：显示答题结果和解析，提供下一题、查看科普、返回首页按钮
- **答题完成界面**：显示成绩信息，提供重新答题、查看排名、返回首页按钮

### 3.4 视觉元素

#### 3.4.1 图标设计
- **方块图标**：64×64像素，卡通化设计，线条简洁流畅
- **按钮图标**：48×48像素，风格统一
- **道具图标**：48×48像素，清晰易懂

#### 3.4.2 动画效果
- **方块动画**：选择、交换、消除、下落动画
- **积分动画**：得分增加时的浮动效果
- **界面动画**：游戏开始、游戏结束、道具使用动画
- **反馈动画**：正确/错误反馈、提示反馈

### 3.5 交互设计
- **触摸交互**：点击、拖拽、滑动操作
- **反馈机制**：音效反馈、视觉反馈、振动反馈
- **响应式设计**：适配不同屏幕尺寸和分辨率

## 四、技术要求

### 4.1 技术栈
- **前端**：Canvas API、JavaScript
- **平台**：微信小游戏

### 4.2 性能要求
- **帧率**：稳定在60FPS
- **加载速度**：游戏启动时间不超过3秒
- **内存占用**：合理控制，避免内存泄漏

### 4.3 代码规范
- **代码结构**：清晰、模块化
- **命名规范**：语义化变量名
- **注释**：完善的代码注释

## 五、资源清单

### 5.1 消消乐游戏
- **图标文件**：15个64×64像素的PNG格式图标（5个非遗、5个自然、5个红色）
- **背景文件**：1个750×1334像素的PNG格式背景图
- **道具图标**：3个48×48像素的PNG格式图标（一键消除、冻结时间、提示功能）
- **按钮图标**：4个48×48像素的PNG格式图标（难度设置、重新开始、结算、返回）
- **音效文件**：5个MP3格式音效（消除、连击、游戏结束、道具使用、升级）

### 5.2 三色拼图游戏
- **拼图图片**：9个750×750像素的PNG/JPG格式图片（3个非遗、3个自然、3个红色）
- **背景文件**：1个750×1334像素的PNG格式背景图
- **按钮图标**：4个48×48像素的PNG格式图标（提示、难度选择、重置、分享）
- **音效文件**：3个MP3格式音效（移动拼图块、拼图完成、提示）
- **打卡海报模板**：1个750×1000像素的PNG格式模板

### 5.3 三色答题游戏
- **题库数据**：3套JSON格式的题库数据（非遗、自然、红色）
- **背景文件**：1个750×1334像素的PNG格式背景图
- **按钮图标**：3个48×48像素的PNG格式图标（提示、提交、跳过）
- **类别图标**：3个48×48像素的PNG格式图标（非遗、自然、红色）
- **音效文件**：5个MP3格式音效（选择选项、提交答案、答题正确、答题错误、提示）
