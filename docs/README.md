# 三色融澄·数字赋能

## 项目简介

"三色融澄·数字赋能"是一款基于微信小游戏平台开发的休闲游戏，包含三色消消乐和三色拼图等游戏模式，旨在通过游戏的方式传播非遗、自然和红色文化。

## 游戏特色

1. **文化融合**：将非遗、自然和红色文化元素融入游戏中
2. **多种游戏模式**：包含消消乐和拼图等多种游戏玩法
3. **成就系统**：通过游戏获得成就，增强游戏趣味性
4. **打卡系统**：支持玩家打卡功能
5. **答题系统**：通过答题了解更多文化知识

## 开发说明

本项目基于微信小游戏平台开发，使用原生JavaScript语言实现。

### 运行环境
- 微信开发者工具
- 微信小游戏平台

### 开发工具
- VS Code
- 微信开发者工具

## 项目结构

项目采用模块化设计，主要分为以下几个部分：

1. **游戏核心**：包含消消乐和拼图游戏的核心逻辑
2. **页面系统**：包含登录页、首页、成就页、打卡页和答题页
3. **数据管理**：负责游戏数据的存储和管理
4. **资源管理**：包含游戏所需的音频、图片等资源

## 如何运行

1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 点击运行按钮即可在模拟器中预览游戏
4. 也可以通过微信开发者工具的真机调试功能在手机上体验游戏
