# 登录页项目简介文案

## 项目名称
三色融澄·数字赋能

## 项目简介
非遗·自然·红色，在游戏中读懂海澄村

## 核心价值
- **趣味传播**：通过游戏化的方式，让海澄村的特色资源变得生动有趣
- **文化传承**：传承和弘扬海澄村的非物质文化遗产
- **生态保护**：宣传海澄村的自然生态环境，增强环保意识
- **红色教育**：传播海澄村的革命历史和红色精神

## 游戏特色
- **三色消消乐**：消除资源，解锁知识
- **三色拼图**：拼出海澄，发现美好
- **三色答题**：答题闯关，成为知识达人
- **线下打卡**：实地探索，解锁专属皮肤

## 项目愿景
让更多人特别是年轻一代了解和关注海澄村的特色资源，促进海澄村的文旅融合发展，为乡村振兴注入新的活力。
