# 三色融澄·数字赋能技术文档

## 1. 项目概述

### 1.1 项目简介
本项目是一个基于微信小游戏框架开发的游戏化小程序，旨在通过游戏化的方式传播海澄村的非遗、自然和红色资源。

### 1.2 技术栈
- **前端框架**：微信小游戏框架
- **游戏引擎**：Canvas API
- **数据存储**：本地存储（wx.setStorageSync/wx.getStorageSync）
- **开发工具**：微信开发者工具

### 2.2 核心模块

#### 2.2.1 应用入口（app.js）
- 负责初始化应用
- 管理页面切换
- 维护全局状态

#### 2.2.2 数据管理器（dataManager.js）
- 管理本地存储
- 数据读写操作
- 数据备份和恢复

#### 2.2.3 页面模块
- **登录页**：微信授权登录
- **首页**：游戏入口和用户信息展示
- **成就页**：成就展示和证书生成

#### 2.2.4 游戏模块
- **消消乐**：6×6/8×8方块矩阵，滑动消除
- **拼图**：9/16/25块拼图，拖动完成
- **答题**：随机抽取题目，判断答案

#### 2.2.5 系统模块
- **成就系统**：成就解锁条件判断，进度追踪
- **打卡系统**：二维码扫描，地理位置验证

## 3. 技术实现

### 3.1 应用架构

#### 3.1.1 页面管理
- 采用单页应用模式
- 通过canvas绘制界面
- 事件监听处理用户交互

#### 3.1.2 数据管理
- 使用微信本地存储存储用户数据
- 数据结构采用JSON格式
- 数据管理器封装数据操作

### 3.2 核心功能实现

#### 3.2.1 微信授权登录
```javascript
wx.getUserInfo({
  success: (res) => {
    GameGlobal.databus.userInfo = res.userInfo;
    GameGlobal.dataManager.saveUserData();
    // 跳转到首页
  }
});
```

#### 3.2.2 消消乐游戏
- **方块矩阵**：6×6或8×8网格
- **滑动逻辑**：监听触摸事件，判断滑动方向
- **消除逻辑**：检测连续相同元素，消除并计分
- **难度梯度**：初级（2种元素）、中级（3种元素）、高级（限时缩短）

#### 3.2.3 拼图游戏
- **图片切割**：将图片切割为9/16/25块
- **拖动逻辑**：监听触摸事件，实现碎片拖动
- **完成检测**：判断所有碎片是否正确拼接
- **打卡海报**：完成后生成可分享的海报

#### 3.2.4 答题系统
- **题库分类**：非遗、自然、红色三类
- **答题逻辑**：随机抽取5道题，判断答案
- **知识达人buff**：连续答对3题，消消乐积分翻倍

#### 3.2.5 成就系统
- **成就分类**：非遗、自然、红色三类
- **解锁条件**：完成特定任务（如消除次数、拼图解锁）
- **奖励机制**：解锁头像框、皮肤，生成数字体验证书

#### 3.2.6 线下打卡系统
- **二维码扫描**：扫描指定二维码
- **地理位置验证**：验证用户是否在海澄村
- **彩蛋解锁**：解锁专属皮肤和彩蛋

### 3.3 性能优化

#### 3.3.1 游戏性能
- 使用对象池管理游戏对象
- 优化Canvas渲染性能
- 减少不必要的计算

#### 3.3.2 页面性能
- 减少DOM操作
- 优化图片资源
- 使用缓存减少重复计算

## 7. 风险与应对

### 7.1 技术风险
- **风险**：游戏逻辑复杂度较高，可能存在性能问题
- **应对**：采用模块化设计，优化游戏算法，进行性能测试

### 7.2 内容风险
- **风险**：海澄村资源资料收集不完整
- **应对**：与当地政府、文化部门合作，确保资料准确性

### 7.3 运营风险
- **风险**：用户获取难度较大
- **应对**：利用微信生态进行传播，开展线上线下活动
