# 海澄村红色资源清单

## 1. 海澄村革命老区
- **简介**：海澄村是珠海市金湾区的革命老区，有着光荣的革命历史和红色传统。
- **历史背景**：在抗日战争和解放战争时期，海澄村人民积极参加革命斗争，为新中国的成立做出了贡献。
- **重要事件**：曾是抗日游击队员的活动据点，多次组织抗日斗争。
- **纪念意义**：是进行爱国主义教育和革命传统教育的重要场所。
- **图片需求**：革命老区标识、历史照片、纪念设施。

## 2. 陈氏宗祠（革命活动旧址）
- **简介**：陈氏宗祠是海澄村的重要历史建筑，也是革命时期的重要活动场所。
- **历史作用**：在革命战争年代，陈氏宗祠曾是地下党组织的秘密联络点和会议场所。
- **建筑特点**：始建于清代，建筑风格独特，保存完好。
- **保护情况**：已被列为金湾区文物保护单位。
- **图片需求**：宗祠外观照片、内部结构、历史文物。

## 3. 海澄村革命烈士纪念碑
- **简介**：为纪念在革命斗争中牺牲的海澄村烈士而建立的纪念碑。
- **历史背景**：纪念碑建于1985年，镌刻着在抗日战争和解放战争中牺牲的烈士姓名。
- **位置**：位于海澄村中心广场，是村民进行爱国主义教育的重要场所。
- **纪念活动**：每年清明节，村民和学生都会到纪念碑前敬献花圈，缅怀先烈。
- **图片需求**：纪念碑照片、纪念活动场景、烈士事迹介绍。

## 4. 红色故事
- **简介**：海澄村流传着许多革命时期的红色故事，展现了村民的革命精神和爱国情怀。
- **代表故事**：
  - 《地下交通员的故事》：讲述了海澄村地下交通员冒着生命危险传递情报的事迹。
  - 《妇女支前队》：讲述了海澄村妇女积极支前，为革命斗争做出贡献的故事。
  - 《儿童团的小英雄》：讲述了海澄村儿童团员协助游击队开展工作的故事。
- **教育意义**：这些红色故事是对青少年进行革命传统教育的生动教材。
- **图片需求**：故事场景插图、相关历史照片。

## 5. 红色文化展览室
- **简介**：海澄村建立了红色文化展览室，展示海澄村的革命历史和红色文化。
- **展览内容**：包括历史照片、革命文物、烈士事迹、红色故事等。
- **开放时间**：每周二至周日开放，免费向公众开放。
- **教育功能**：是海澄村开展爱国主义教育和革命传统教育的重要基地。
- **图片需求**：展览室外观、内部展览、参观场景。
